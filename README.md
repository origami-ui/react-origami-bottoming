
Bottoming Component
----

Demo http://origami-ui.org/react-origami-bottoming/

### Usage

```
npm i --save react-origami-bottoming
```

```coffee
Bottoming = React.createFactory require :react-origami-bottoming

Bottoming null
  childrenKeepGrowing
```

This component will help scrollbar stick to the bottom.
